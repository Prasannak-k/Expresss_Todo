# To-Do Manager

A full-stack To-Do Manager built with Express.js and a vanilla JavaScript frontend. This project allows users to create, read, update, and delete daily tasks, with all data saved to a local JSON file.

## Features
* **Create:** Add new tasks to the list.
* **Read:** View all pending and completed tasks.
* **Update:** Edit task descriptions or mark them as done/pending.
* **Delete:** Remove tasks entirely.

## Tech Stack
* **Backend:** Node.js, Express.js
* **Frontend:** HTML, CSS, Vanilla JavaScript
* **Database:** Local JSON file (`mock_data.json`)
