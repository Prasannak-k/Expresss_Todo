# To-Do Manager (MongoDB Edition)

A full-stack, RESTful task management application built to demonstrate backend API development, database integration, and dynamic frontend rendering. 

## Features
* **Full CRUD Operations:** Create, Read, Update, and Delete tasks seamlessly.
* **Persistent Storage:** Tasks are securely stored and retrieved using MongoDB.
* **Dynamic UI:** Real-time frontend updates using vanilla JavaScript and DOM manipulation.
* **Search & Filter:** Instantly filter tasks by name using the built-in search bar.

##  Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **Environment Management:** dotenv
